# Interactive Boxes Task

This is a solution to the **Front End Technical Task** involving interactive boxes using only basic HTML, CSS, and JavaScript (no libraries/frameworks used).

## 🔧 Features
- Pure HTML + CSS + Vanilla JS (No frameworks)
- 3 Interactive boxes (Top, Middle, Bottom)
- Expandable sections when clicked
- Dropdowns to change box **color** and **size**
- Fully responsive and styled with custom CSS

## 📦 How to Deploy on GitHub Pages
1. Push this project to GitHub
2. Go to your repository > Settings > Pages
3. Under "Source", select `main` branch and `/root` folder
4. GitHub will give you a link after deployment (within seconds)
